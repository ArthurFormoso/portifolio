Sobre o Projeto

Este projeto é um site desenvolvido utilizando Python com o framework Flask. O intuito do projeto era desafiar os alunos a criar um site com a linguagem de programacao Python, aplicando conceitos de desenvolvimento web, roteamento e templates.


O site apresenta uma interface simples e funcional, permitindo aos usuários navegar por diferentes páginas e interagir com o conteúdo. Durante o desenvolvimento foram abordados tópicos como configuração do ambiente Flask, criação de rotas, manipulação de dados e renderização de templates HTML.


Não funciona em um computador comum devido a linguagem e framework Flask.
